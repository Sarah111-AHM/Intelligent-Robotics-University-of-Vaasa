## Intelligent Robotics – Course Overview (ICAT3110-3012)

This repository contains the summary and materials for Lecture 1: Introduction to the Intelligent Robotics course taught by Mahmoud Elsanhoury at the University of Vaasa.

Course Objectives

By the end of this course, students will be able to:

· Understand the fundamentals of robotics and autonomy.
· Build and propose intelligent robotic solutions for complex tasks.
· Use simulation platforms for hands-on practice.
· Identify challenges faced by intelligent robots.

Key Concepts Introduced

· Robot: Programmed machine for complex tasks, human-supervised.
· Automatic vs. Autonomous:
  · Automatic: Pre-programmed, no decision-making.
  · Autonomous: Perceives, analyzes, reasons, and decides independently.
· Every autonomous system is automatic, but not vice versa.

Required Components for an Intelligent Robot

· Movement tools (wheels, legs, etc.)
· Locomoted arms (lift, suction, etc.)
· Vision system (cameras, flash, multi-cam)
· Power source (wired, battery, solar)
· Sensors (force torque, collision, part detection)
· Memory storage (built-in, external, mass storage)
· Processing unit (microcontroller MCU or microprocessor MPU)

Real-World Examples & Warnings

· Challenger Mission 1986 – failure due to inaccurate/unreliable data.
· Accurate data enables intelligent behavior (videos shown in class).
